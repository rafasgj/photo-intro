\begin{frame}
  \frametitle{Adobe Lightroom Develop}
  \begin{itemize}
      \item O módulo \textit{Develop} do Lightroom é uma interface para o Adobe Camera Raw.
      \item Os mesmos comandos podem ser encontrados no Adobe Camera Raw no Photoshop.
      \item A grande diferença, é a possibilidade de automações no Lightroom.
      \item A grande vantagem de utilizar o Lightroom é a integração dos seus módulos.
  \end{itemize}
\end{frame}

            \end{itemize}
        }

\begin{frame}
    \frametitle{\textit{Softproofing}}
    \begin{columns}
        \column{0.3\textwidth}{
            \begin{tikzpicture}
                \node[anchor=south west,inner sep=0] at (0,0)
                    {\includegraphics[width=4.3cm]{images/lr_soft_proofing_rgb.png}};
                \draw[cyan,ultra thick,rounded corners]
                    (0.1, 1.85) rectangle (4.2, 2.15);
            \end{tikzpicture}
        }
        \column{0.65\textwidth}{
            \begin{itemize}
                \item \textit{Softproofing} permite prever como ficará a imagem
                de saída, impressa ou em exibição.
                \item Os avisos de brancos e pretos se alteram para mostrar quais
                cores não são corretamente reproduzidas.
                \item Sempre utilize a simulação de papel e tinta.
                \item Os valores RGB exibidos são os valores absolutos, sob o
                ponteiro do \textit{mouse}.
            \end{itemize}
        }
    \end{columns}
\end{frame}
